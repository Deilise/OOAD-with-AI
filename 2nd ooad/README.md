# 2nd OOAD

This folder is an organized copy of the Roomba RVC OOAD artifacts.

## Folder Map

- `01_requirements/` - Software Requirements Specification (SRS)
- `02_use_cases/` - Use case document
- `03_diagrams/class/` - Class diagram source, Markdown, and rendered image
- `03_diagrams/domain/` - Domain diagram source, Markdown, and rendered image
- `03_diagrams/ssd/` - System Sequence Diagrams (SSD) by use case
- `03_diagrams/sd/` - Sequence Diagrams (SD) by use case
- `04_code/` - C++ source, headers, example, and CMake project file
- `05_unit_tests/` - C++ unit test files
- `06_simulator/` - Simulator source and run scripts

## Main Entry Points

- SRS: `01_requirements/RVC_SW_Controller_SRS.md`
- Use cases: `02_use_cases/RVC_SW_Controller_UseCases.md`
- Domain diagram: `03_diagrams/domain/RVC_Domain_Diagram.md`
- SSD index: `03_diagrams/ssd/RVC_SSD_Index.md`
- SD index: `03_diagrams/sd/RVC_SD_Index.md`
- Class diagram: `03_diagrams/class/RVC_Class_Diagram.md`
- Code summary: `04_code/RVC_Code.md`
- Unit test summary: `05_unit_tests/RVC_Unit_Tests.md`
- Simulator summary: `06_simulator/RVC_Simulator.md`
